# O Captain Admissions Website

This folder is ready for GitHub Pages deployment.

## Deploy on GitHub Pages

1. Create a new GitHub repository, for example `o-captain-admissions`.
2. Upload all files in this folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. Your website will be available at:

```text
https://<your-github-username>.github.io/o-captain-admissions/
```

If you use a different repository name, replace `o-captain-admissions` with your actual repository name.
